My First Pet Project - Weather Forecast

Hello everyone! Today, I'm excited to present to you my first pet project - Weather Forecast.Let's dive in!

Project Overview:

  • Name: Weather Forecast
  • Description: Weather Forecast is a web application that provides real-time weather information for location.
  • Technologies Used: Spring Boot, Bootstrap, JavaScript, CSS, HTML

Key Features:

  1. Accurate Weather Data:
    • The application utilizes the OpenWeather free API to fetch up-to-date weather information for any desired location.
    • Users can get current weather conditions and temperature.
  2. User-Friendly Interface:
    • The application has an intuitive and responsive user interface designed using Bootstrap, ensuring a seamless experience across devices.
  3. Location-Based Forecast:
    • Users can enter their preferred location to get weather forecasts specific to that area.
    • The application dynamically updates the weather information as per the user's selected location.

Technical Stack:

  • Spring Boot: serves as the backend framework handling API integration, data retrieval, and business logic implementation.
  • Bootstrap, JavaScript, CSS, HTML: used for creating a visually appealing and responsive user interface.

In conclusion, the Weather Forecast project is my first own pet project that allowed me to apply my skills in Spring Boot, Bootstrap, JavaScript, CSS, and HTML to create a user-friendly weather application.
